# Collaborative UI Library

`@jsonjoy.com/collaborative-ui` is a library of React components designed for
building rich real-time collaborative editing experiences.


## Usage

Installation:

```
npm install json-joy @jsonjoy.com/collaborative-ui
```

Usage:

```tsx
import {MyComponent} from '@jsonjoy.com/collaborative-ui/lib/MyComponent';

<MyComponent {...} />
```
